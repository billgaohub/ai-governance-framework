## Three Core Pillars

### 1. 决策权守恒 / Decision Rights Conservation

> AI 只是认知放大器，人类始终保持最终决策权。

- 系统可以放大认知，但不得收缩选择空间
- 所有建议必须可追溯、可解释、可反转
- System amplifies cognition; never shrinks the option space
- All recommendations: traceable, explainable, reversible

### 2. 三层认知控制 / Three-Layer Cognitive Control

| 层级 Layer | 控制点 Control | 机制 Mechanism |
|-----------|---------------|---------------|
| 输入控制 Ingest | 假设≠事实 Assumption≠Fact | 推断必须标记验证 / Inferences marked for verification |
| 上下文控制 Context | 认知必须可压缩 Compressible | 动态遗忘/合并 / Dynamic forget/merge |
| 推理控制 Reasoning | 上下文受预算 Budget | 动态裁剪 / Dynamic pruning |

### 3. 渐进演化 / Gradual Evolution

- 每日复盘 / Daily review — 扫描偏离度 / Scan deviation
- 成功模式固化 / Success pattern hardening
- 保守但持续 / Conservative but continuous improvement
