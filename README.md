# AI Governance Framework

> 企业级 AI 治理框架：让 AI 系统可控、可审计、可演化

---

## 核心原则

### 1. 决策权守恒
**AI 只是认知放大器，人类始终保持最终决策权。**

- 系统可以放大认知，但不得收缩选择空间
- 所有建议必须可追溯、可解释、可反转

### 2. 三层认知控制
| 层级 | 控制点 | 核心机制 |
|------|--------|----------|
| 输入控制 | Ingest | 假设≠事实，推断必须标记验证 |
| 上下文控制 | Context | 认知必须可压缩，动态遗忘/合并 |
| 推理控制 | Reasoning | 上下文受预算约束，动态裁剪 |

### 3. 渐进演化
- 每日复盘，扫描偏离度
- 成功模式固化
- 保守但持续的自我改进
